## Overview

The `MatchaTTS` class is the core model that implements fast text-to-speech synthesis using conditional flow matching. It combines a text encoder, duration predictor, and flow matching decoder to generate high-quality mel-spectrograms from text.

## Class Definition

## Methods

### synthesise()

Generates mel-spectrogram from text input.

```python
@torch.inference_mode()
def synthesise(
    x: torch.Tensor,
    x_lengths: torch.Tensor,
    n_timesteps: int,
    temperature: float = 1.0,
    spks: torch.Tensor = None,
    length_scale: float = 1.0
) -> dict
```

#### Parameters

<ParamField path="x" type="torch.Tensor" required>
  Batch of texts converted to phoneme embedding IDs
  
  **Shape:** `(batch_size, max_text_length)`
</ParamField>

<ParamField path="x_lengths" type="torch.Tensor" required>
  Lengths of texts in batch
  
  **Shape:** `(batch_size,)`
</ParamField>

<ParamField path="n_timesteps" type="int" required>
  Number of ODE solver steps for flow matching. Higher values produce better quality but slower synthesis. Typical range: 4-20
</ParamField>

<ParamField path="temperature" type="float" default="1.0">
  Controls variance of the terminal distribution. Higher values increase diversity but may reduce quality. Range: 0.0-2.0
</ParamField>

<ParamField path="spks" type="torch.Tensor" default="None">
  Speaker IDs for multi-speaker models
  
  **Shape:** `(batch_size,)`
</ParamField>

<ParamField path="length_scale" type="float" default="1.0">
  Controls speech pace. Values greater than 1.0 slow down speech, less than 1.0 speed up speech
</ParamField>

#### Returns

<ResponseField name="encoder_outputs" type="torch.Tensor">
  Average mel-spectrogram generated by the encoder
  
  **Shape:** `(batch_size, n_feats, max_mel_length)`
</ResponseField>

<ResponseField name="decoder_outputs" type="torch.Tensor">
  Refined mel-spectrogram improved by the CFM decoder
  
  **Shape:** `(batch_size, n_feats, max_mel_length)`
</ResponseField>

<ResponseField name="attn" type="torch.Tensor">
  Alignment map between text and mel-spectrogram
  
  **Shape:** `(batch_size, max_text_length, max_mel_length)`
</ResponseField>

<ResponseField name="mel" type="torch.Tensor">
  Denormalized mel-spectrogram ready for vocoder
  
  **Shape:** `(batch_size, n_feats, max_mel_length)`
</ResponseField>

<ResponseField name="mel_lengths" type="torch.Tensor">
  Actual lengths of generated mel-spectrograms
  
  **Shape:** `(batch_size,)`
</ResponseField>

<ResponseField name="rtf" type="float">
  Real-time factor (lower is faster). RTF < 1.0 means faster than real-time
</ResponseField>

#### Example

```python
import torch
from matcha.models.matcha_tts import MatchaTTS
from matcha.text import text_to_sequence
from matcha.utils.utils import intersperse

# Load pretrained model
model = MatchaTTS.load_from_checkpoint("matcha_ljspeech.ckpt")
model.eval()

# Prepare text input
text = "Hello, this is a test."
sequence = text_to_sequence(text, ["english_cleaners2"])[0]
x = torch.LongTensor(intersperse(sequence, 0)).unsqueeze(0)
x_lengths = torch.LongTensor([x.shape[1]])

# Generate mel-spectrogram
output = model.synthesise(
    x=x,
    x_lengths=x_lengths,
    n_timesteps=10,
    temperature=0.667,
    length_scale=1.0
)

print(f"Generated mel shape: {output['mel'].shape}")
print(f"Real-time factor: {output['rtf']:.4f}")
```

  Length of segment to cut for decoder training. Should be divisible by 2^(num_downsamplings)

<ParamField path="cond" type="torch.Tensor" default="None">
  Additional conditioning (reserved for future use)
</ParamField>

<ParamField path="durations" type="torch.Tensor" default="None">
  Precomputed durations when `use_precomputed_durations=True`
</ParamField>

#### Returns

<ResponseField name="dur_loss" type="torch.Tensor">
  Duration prediction loss
</ResponseField>

<ResponseField name="prior_loss" type="torch.Tensor">
  Prior loss between encoder outputs and target mel-spectrogram
</ResponseField>

<ResponseField name="diff_loss" type="torch.Tensor">
  Flow matching loss from the decoder
</ResponseField>

<ResponseField name="attn" type="torch.Tensor">
  Attention alignment computed by Monotonic Alignment Search
  
  **Shape:** `(batch_size, max_text_length, max_mel_length)`
</ResponseField>
